## 1. Scope

Processor provides hosted or self-hosted RAG services for customer internal documentation search and Q&A.

## 2. Processing Details

- Subject matter: enterprise knowledge retrieval
- Duration: agreement term + retention period
- Nature: storage, indexing, retrieval, and response generation
- Data subjects: customer employees/authorized users
- Personal data: user account metadata and any personal data contained in uploaded content

## 3. Processor Obligations

- process personal data only on documented instructions
- ensure confidentiality obligations for personnel
- implement technical and organizational security measures
- assist controller with data subject rights and DPIA requests
- notify controller of personal data breach without undue delay

## 4. Subprocessors

Processor may use subprocessors listed in a maintained subprocessor list. Processor will notify Controller before adding new subprocessors where contractually required.

## 5. International Transfers

Default deployment is EU-hosted. Any non-EU transfer requires Controller approval and a valid transfer mechanism.

## 6. Deletion and Return

Upon termination, Processor will delete or return personal data as instructed by Controller, except where retention is required by law.

## 7. Audits

Controller may request reasonable evidence of compliance and security controls no more than once annually, subject to confidentiality safeguards.
